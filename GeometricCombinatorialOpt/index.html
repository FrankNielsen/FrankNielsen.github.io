Geometric combinatorial optimization

I am interested in geometric combinatorial optimization, specially geometric Set Cover Problems (SCPs) or Hitting Set Problems (HSPs). Those problems are NP-hard: For example, stabbing or piercing a set of isothetic boxes or finding maximum independent set of geometric graphs. A recent breakthrough was obtained by Joseph S. B. Mitchell here.
