  • Editors: Frederic Barbaresco and Frank Nielsen

  • Please use the following templates:
    • LaTeX: contributed-books.zip (Springer svmult.cls templates)
    • MS Word
    • SPIGL copyright form

    Submit your chapter submission to Easychair as follows:

    1. Be sure to include all LaTeX sources (macros, figures) so we can recompile the chapter. Includes also the compiled PDF of your chapter so that we can compare. If you use images from third parties, please get the necessary copyright approvals.
    2. Fill, sign and scan/save the PDF of SPIGL copyright form (here is the copyright form in PDF)
    3. Create a zip file SPIGL-number-contactauthor.zip (see https://easychair.org/conferences/?conf=spigl2021 for your submission number) and send it by email to nielsen@lix.polytechnique.fr (CC: frederic.barbaresco@thalesgroup.com) with email title:[SPIGL] number contactauthor (for example, [SPIGL] 3 nielsen)

    • Deliver to springer: 30th April 2021
    • Available : Summer 2021 (before GSI 2021)
